# gtopt Marketing and Outreach Materials

This directory contains promotional and informational materials for the
gtopt project.

## Contents

| File | Description | Audience |
|------|-------------|----------|
| `project-overview.md` | One-page project summary with feature table | General |
| `technical-brief.md` | Technical highlights for power system engineers | Engineers |
| `comparison-summary.md` | Comparison with PyPSA, GenX, PLP, pandapower | Evaluators |

## Usage

These materials can be:
- Shared as standalone documents
- Used as content for conference posters or presentations
- Adapted for grant proposals
- Posted on social media or project websites
